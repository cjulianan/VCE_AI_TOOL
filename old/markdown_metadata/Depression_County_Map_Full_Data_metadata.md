# Depression_County_Map_Full_Data.csv

**Description:** Tracking number of people who report having depression grouped by county and per 100k people

- **Source Organization:** Mental Health America
- **Geographic Coverage:** county
- **Temporal Coverage:** 2020-2025
- **URL Source:** [Link](https://mhanational.org/data-in-your-community/mha-state-county-data/)
- **Spatial Alignment:** FIPS Column: `fips_code`, Locality Column: `County Name`

### Column Schema

* **`County Name`** (*string*): The name of the local county geographic area. *(Keywords: county, location, area, region, place)*
* **`fips_code`** (*integer*): Federal Information Processing Series unique numerical code identifier for the county. *(Keywords: fips, code, county code)*
* **`State`** (*string*): The name of the US state where the county is located. *(Keywords: state)*
